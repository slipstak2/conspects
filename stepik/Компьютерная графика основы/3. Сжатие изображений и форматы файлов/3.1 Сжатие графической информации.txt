Алгоритм != Формат
Алгоритм Jpeg != Формат Jpeg
Метод более общее понятие чем алгоритм


Алгоритмы сжатия изображений
---------------------------------
Сжатие бывает:
   - без потерь (lossless)
   - с потерями (lossy)
   
RLE
   - Ориентирован на однобитные изображения
   - Работает по строкам

LZW(сжатие x4-100(7мб одноцветного изображения))
   - Применим к любыми типам данных
   - Ориентирован на работу с индексированным цветом
   
Lossless JPEG (сжатие x2-20)
- Для полноцветных изображений

Вейвлет и фрактальное сжатие
------------------------------------
Вейвлет преобразование (волновое, рекурсивное)

Цепочка преобразований JPEG
   RGB -> YCbCr
   Субдисретизация
   ДКП(дискретное косинусное преобразование)  -> меняем на Вейвлет
   Квантование
   Линейный вид
   RLE
   Хаффман
   
- Нет дробления на блоки(нет эффекта Гиббса)
- Постепенное декодирование(удобно передавать по сети)
- Наличие миниатюры изображения
- Настраивается степень сжатия (2x - 200x)
- Артефакты сжатия появляются на контурах(при больших коэффициентах сжатия)
- Основной алгоритм сжатия в JPEG2000

Фрактальное сжатие
------------------------------------
- Оптимален для сжатия самоподобных изображений
- Коэффициент сжатия до 20000 раз
- На обычных изображениях тоже сжимает лучше других алгоритмов
- Крайне медленное кодирование
- Декодирование быстрее чем JPEG
- Избегает пикселизации при масштабировании (до 3-4 раз)
