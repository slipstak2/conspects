Форматы графических файлов

Графический формат - способ организации информации об изображении в графическом файле
Графический формат определеляет:
   Способ храненния информации в файле:
      - растровый
      - векторный
      - смешаный
      - двухмерный
      - трехмерный
   Форма хранения информации
      - Алгоритм сжатия
      
2D             Универсальные                 Оригинальные
Растровые      BMP, GIF, JPG, JPEG2000       PSD
               PNG, TIFF, WebP (RAW)

Векторные      WMF, EMF, PS, EPS,            AI CDR, CMX, FLA
               SVG, SVGZ                     SWF, DWG

Документы      PDF                           DJVU


Растровые графические форматы:
------------------------------------
BMP
Состав:
   -Заголовок
   -Каталог
   -Изображения
Цвет:
   - 1...24 бит/пиксель
   - только RGB
   - нет прозрачности
Сжатие:
   - RLE, отключаемое
   
GIF
   Сжатие:
   - LZW
   Цвет:
   - Key Color(1 цвет). Палитка из 256 цветов. Поддержка прозрачности
   Возможности:
   - Анимация
   Применение:
   - Текст в графике, схемы(для малоцветных изображений критичные к потере четкости контуров, т.к. в нем применяется сжатие без потерь) Если нужно представить текст картинкой, схему проезда - это лучший вариант
   - Анимированые банеры и картинки для веб
   
JPEG
   Сжатие:
   - JPEG
   Цвет:
   - до 32 бит/канал
   - нет прозрачности
   - разные цветовые модели
   Возможнсоти:
   - метаданные
   Применение:
   - Фотоизображения для просмотра
   - Полноцветная графика в веб
   
JPEG2000
   Ограничения:
   - Слабая поддержка воспроизведения
   Сжатие:
   - вейвлет преобразование
   
PNG
   Сжатие:
   - Недеструктивное(lossless)
   Цвет:
   - до 32 бит/канал
   - канал прозрачности(альфа)
   Возможности:
   - метаданные
   - разметка для нарезки веб-страниц
   Анимация:
   - расширение формата (APNG)
   - и отдельный формат(MNG)
   
TIFF
   Предназначение:
   - Профессиональная полиграфия
   Цвет:
   - Различные цветовые модели
   - Цветовые профили
   - Многослойная прозрачность(маски)
   Возможности:
   - Запись неграфической информации(звук)
   - Многостраничность(ограниченная)
   Сжатие:
   - Недеструктивное: ZIP, LZW
   - Деструктивное: JPEG
   
   
(RAW) - Семейство форматов фото-негативов
   High Dynamic Range(HRD)
   Особенности:
   - Нужно импортировать снимок
   - Это "сырое" изображение.
     Камера не корректирует:
     - баланс белого
     - смещение экспозиции(AE Shift)
     - параметры экспозиции и цвета
     - резкость и шумоподавление
   - Изображения не сжаты
   
WebP(от Google)   Статические изображения (Web Pictures)
WebM              Видео(Web Motion)
   Назначение:
   - Веб-сайты
   Поддержка:
   - Браузеры Chrome, Opera
   - ОС Android
   - Остальные браузеры - через Flash и JavaScript
   Цвет:
   - 8 бит/канал
   - цветовая субдискретизация 4:2:0
   
PSD
   Назначение:
   - Рабочий формат Adrom PhotoShop
   Возможности:
   - Поддержка слоев и масок прозрачности
   - Работа с текстом
   - Работа с векторными объектами
   - Работа с 3D
   - Работа с видео
   
   
   
Векторные графические форматы
------------------------------------
WMF      Windows MetaFile
EMF      Enchanced MetaFile

PS       PostScript(Adobe). Язык общения с принтером
         Применение:
         - Команды для принтера
         - Файл для печати
         - Для PS-совместимой техники

EPS      Разновидность PS
         Свойства:
         - Аппаратная независимость
         - Одностраничный
         - Применим для обмена изображениями

SVG      Scalable Vector Graphics
         - Основной векторный формат в веб
         - Поддерживает анимацию
         - Отслеживает события на странице
         - Открытый
         - Сжатая версия SVGZ компактнее
         
Векторные редакторы
------------------------------------
Adobe Illustrator (.ai)
   - Удобен для графического дизайна
   - Надежно работает с цветом
CorelDraw (.cdr) (.cmx)
   - Импорт/экспорт большого числа форматов
   - Удобнее в допечатной подготовке
Adobe Flash (.fla(рабочий формат), .swf)
   - графика, анимация, интерактив
   - перестал поддерживаться а на мобильных платформах
Autodesck Autocad(.dwg)
   - основной формат для чертежей 2D и 3D

Форматы документов
------------------------------------
PDF
   - Создается многими программами
   - Верстка и цвета передаются без искажений
   - Способ посмотреть .ps файлы (через Distiller)
   - Электронная распечатка
   - Редактируемый (через Acrobat и не только)
   - Поддерживает интерактивные элементы
   - Поддерживает звук и видео
   - Встроенные средства шифрования
   
DjVu
   - Программно зависим
   - Сжатие варьируется от типов данных
   - При этом, кодировщик работает на входе с растровыми изображениями
