Переменные(variable)
   именованная область памяти

имя(name):
   буквы, цифры, _;
   не может начинаться с цифры.
   
значение(value):
   числа;
   строки(если есть пробелы, то в кавычках)
   отдельные символы
   
Запись(перезапись) значения:
   <имя>=<значение> (около равно без пробелов)
   path=~/Docs

Чтение:
$<имя> или ${<имя>}

path2=$path/file.txt    # path2=~/Docs/file1.txt
echo "Path is $path2"   # Path is ~Docs/file1.txt
echo "Path is ${path}2" # Path is ~/Docs2


#!/bin/bash

var1="Hello, world!"
var2="var1 is $var1"

echo '$var2'         # $var2
echo "$var2"         #  var1 is Hello, world!
echo "\$var2"        # $var2


Аргументы
Передача аргументов скрипту:
./script.sh arg1 arg2 arg3 ...

Обработка внутри скрипта:
$1 первый аргумент
$2 второй аргумент
...
$0 имя скрипта
$# количество аргументов
