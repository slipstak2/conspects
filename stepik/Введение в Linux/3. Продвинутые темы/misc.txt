Права доступа:
   Узнать кто сейчас в системе:
   users
   
   Узнать какие вообще есть пользователи:
   less /etc/passwd
      syslog:x:100:103::...
      bi:x::1000:1000:bi,,,... (UID >=1000)
      
   Узнать группы пользователя:
   groups <имя_пользователя>
   
   Узнать права доступа:
   ls -l <путь>
   
   drwxrwxr-x  2  user  group ...   dir
   -rw-rw-r--  1  user  group ...   file
   
   Первая буква - d для папок. Остальные 9 нужно разбить на тройки:
   
   rw- (110)      owner                   r(100, 4) - Read
   rw- (110)      group                   w(010, 2) - Write
   r-- (100)      other                   x(001, 1) - eXecute

   Общие права доступа = 664

   Права доступа у файлов:
      r - просмотр содержимого
      w - редактирование
      x - запуск
      
   Права доступа у директорий:
      r - просмотр содержимого
      w - создание/удаление файлов/поддиректорий
      x - 1) вход в директорию (через cd)
          2) просмотр inode файлов/поддиректорий
          

    Изменить права доступа:
      chmod [ugoa] [+-][rwx] <путь>
      chmod ug+rw file.txt
      
      u - user
      g - group
      o - others
      a - all
      
      chmod [octal code] <путь>
      chmod 777 file.txt
      
   Изменить обладателя:
      chown new_user:new_group <путь>
      
      chown zoidberg    file.txt
      chown :futurama   file.txt
      
   Суперпользователь:
   root (группа root)
   
   Сделать от имени root:
   sudo <действие>
   sudo mkdir dir
   sudo chmod -r dir
   
   sudo - substitute user and do (подменить пользователя и выполнить)
   
   Новые команды:
   Посчитать что-то в файле:
   wc [что-считаем] <путь>
   wc -l file.txt (количество строк)
   wc -w file.txt (количество слов)
   wc -c file.txt (количество символов)
   
   Сравнить файлы/директории:
   diff [-q -r] <путь1> <путь2>
   diff file1.txt file2.txt | less
   diff -qr dir1/ dir2/
   
   -r - рекурсивно проходится внутри и сравнивает имена с одинаковыми именами
   -q - не выводить разницу, а выводить только факт
   
   Узнать сколько места занимаем на диске:
   du [--max-depth <глубина> -h] <путь>
   du -h ~/
   du --max-depth 1 -h .
   
   df [-h]
      узнать сколько места занято/свободно на всей системе
      
   Вывести часть файла:
   head [-n <количество строк>] <путь>
   tail [-n <количество строк>] <путь>
   
   head -n 10 file.txt
   tail -n 50 file.txt | less
   
   Работа с файлами/директориями:
   можно использовать {имя1, имя2, ...}
   
   mkdir ./{dir1,dir2,dir3}  Нельзя ставить пробел после запятой
   touch ~/{file1, file2}.txt
   
   Вывод с сортировкой:
   ls --sort=[вид сортировки] -l <путь>
   ls --sort=size -l ~/
   ls --sort=time -l .
   
   Перенаправление в один файл:
   &> или &>>
   program &>> both.txt
   
   Перенаправление одного потока в другой:
   2>&1  stderr в stdout
   1>&2  stdout в stderr
   
   Перенаправление в никуда и из ниоткуда:
   /dev/null
   programm 2>/dev/null
   cat /dev/null > file.txt
